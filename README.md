# ✨ සිහිනය - Sihinaya ✨

> *"සිහිනයක් තරම් මෘදු වෙබ් අඩවියක්..."*

---

## 🌙 විස්තරය | Description

**සිංහල:**  
"සිහිනය" යනු සිංහල භාෂාවේ රසය, ආදරය, සහ කාව්‍යමය හැඟීම් එකතු කරන ලද **Hobby Web Project** එකකි. මෙය HTML, CSS, සහ Vanilla JavaScript භාවිතයෙන් හදවතින්ම නිර්මාණය කර ඇත.

**English:**  
"Sihinaya" (meaning "Dream") is a beautifully crafted **hobby web project** built with pure HTML, CSS, and Vanilla JavaScript. It brings together the beauty of the Sinhala language, love, and poetic emotions.

---

## ✨ විශේෂාංග | Features

- 🌟 **Floating Hearts Animation** – හදවත් පාවෙන ලස්සන ඇනිමේෂන්
- 💖 **Interactive Magic Button** – හදවත ස්පර්ශ කරන බොත්තමක්
- 🌙 **Poetic Cards** – සඳ, තරු, මැණික යන තේමා සහිත කාඩ් තුනක්
- 🎨 **Modern UI** – Glassmorphism Design, Gradient Text, Smooth Animations
- 💫 **Responsive** – සියලුම තිර ප්‍රමාණවලට ගැලපෙන නිර්මාණයක්
- 💬 **Sinhala Language** – සම්පූර්ණයෙන්ම සිංහලෙන්

---

## 🛠️ භාවිතා කළ තාක්ෂණයන් | Tech Stack

| Technology | Purpose |
|------------|---------|
